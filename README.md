## ‚ú® Features

- **Instant Control** ‚Äî Start, Stop, and Restart Valet services (Nginx, PHP, DnsMasq) in one click.
- **Site Management** ‚Äî View all your parked `.test` domains with live SSL status indicators.
- **Native Experience** ‚Äî Built with SwiftUI and `NSVisualEffectView` for that premium macOS feel.
- **Quick Actions** ‚Äî Instant browser navigation for any local site.

## üõ† Under the Hood

ValetBar is built with **Swift 6.2.3** and **SwiftUI**, adhering to the highest standards of macOS development.
It includes **Sparkle 2** for secure, sandboxed auto-updates and is fully signed with the **Hardened Runtime**.
